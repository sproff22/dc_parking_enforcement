# DC Parking Enforcement UI

This is the dashboard frontend for our DC parking enforcement project. **This is a draft** – we are currently working on data collection, cleanup, and modeling and the UI is powered by mock data.

## Running the application

The user interface lives in this `ui/` directory and is built with Vite + React.
To run it locally:

```bash
cd ui
npm install        # install dependencies (requires Node.js/npm)
npm run start      # or `npm run dev`
```

The development server will start (usually at http://localhost:5173) and automatically reload as you edit source files.
